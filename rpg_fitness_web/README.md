# 🎮 RPG Fitness Tracker — Web Interface

Красивый веб-интерфейс с плавными анимациями для вашей фитнес-RPG системы.

## 🚀 Быстрый запуск

### 1. Запуск бэкенда (Terminal 1)

```bash
cd /workspace/rpg_fitness_web/backend
python api.py
```

Сервер запустится на `http://localhost:8000`

### 2. Запуск фронтенда (Terminal 2)

**Вариант A: Простой HTTP сервер**
```bash
cd /workspace/rpg_fitness_web/frontend
python -m http.server 3000
```

**Вариант B: Live server с авто-обновлением**
```bash
npm install -g live-server
live-server frontend --port=3000
```

Откройте браузер: **`http://localhost:3000`**

## ✨ Фичи интерфейса

### Анимации
- **Плавное появление** карточек при загрузке
- **Level-up анимация** при повышении уровня (пульсация + масштаб)
- **Прогресс-бары** с плавным заполнением (1 секунда)
- **Ховер-эффекты** на карточках (подъём + увеличение)
- **Уведомления** с анимацией въезда/выезда
- **Спиннер загрузки** с вращением

### Визуальные эффекты
- **Glassmorphism** дизайн (полупрозрачные карточки)
- **Градиентный фон** (тёмная тема)
- **Цветовая кодировка** зон тела
- **Тени с свечением** для важных элементов
- **Кастомные скроллбары**

### Интерактивность
- **Авто-обновление** данных каждые 30 секунд
- **Мгновенная обратная связь** после записи тренировки
- **Переключение вкладок** без перезагрузки
- **Выбор упражнения** с предпросмотром зон
- **Слайдер интенсивности** с визуальной обратной связью

## 📊 Структура приложения

```
RPG Fitness Tracker
├── Backend (FastAPI)
│   ├── GET /api/progress — Прогресс пользователя
│   ├── POST /api/workout — Запись тренировки
│   ├── GET /api/exercises — Список упражнений
│   ├── GET /api/zones — Список зон
│   └── GET /api/stats — Статистика
│
└── Frontend (React + Framer Motion)
    ├── Dashboard — Общий уровень + зоны
    ├── Workout — Форма записи тренировки
    └── Exercises — База упражнений
```

## 🔧 Настройка

### Изменение скорости анимаций

В `frontend/index.html` найдите:
```css
.xp-bar-fill {
    transition: width 1s cubic-bezier(0.4, 0, 0.2, 1); /* Измените 1s */
}
```

### Изменение цветов

В секции `<script>` найдите `ZONE_COLORS` и измените hex-коды.

### Настройка API URL

Если бэкенд на другом порту:
```javascript
const API_URL = 'http://localhost:8000'; // Измените порт
```

## 🛠 Расширение

### Добавить новую анимацию

```jsx
<motion.div
    initial={{ opacity: 0, x: -50 }}
    animate={{ opacity: 1, x: 0 }}
    transition={{ duration: 0.5 }}
>
    {/* Контент */}
</motion.div>
```

### Добавить новый компонент

Создайте функцию-компонент и используйте Framer Motion:

```jsx
const NewComponent = ({ data }) => {
    return (
        <motion.div
            whileHover={{ scale: 1.05 }}
            className="..."
        >
            {/* ... */}
        </motion.div>
    );
};
```

## 🐛 Решение проблем

### CORS ошибки
Убедитесь, что бэкенд имеет CORS middleware:
```python
app.add_middleware(CORSMiddleware, allow_origins=["*"], ...)
```

### Данные не обновляются
Проверьте, что бэкенд запущен на порту 8000:
```bash
curl http://localhost:8000/api/progress
```

### Анимации тормозят
Уменьшите количество одновременно анимируемых элементов или упростите effects.
