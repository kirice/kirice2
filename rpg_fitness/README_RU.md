# 🎮 RPG Fitness Tracker — Полное руководство

## 📖 Описание
Геймифицированный фитнес-трекер с RPG-системой уровней, интерактивной моделью тела и умным расчётом прогресса.

---

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
pip install flet
```

### 2. Запуск приложения

#### Windows:
Дважды кликните на **`run_windows.bat`** или:
```bash
python gui_app.py
```

#### Linux/Mac:
```bash
./run.sh
# или
python gui_app.py
```

Приложение откроется автоматически в вашем браузере по адресу `http://localhost:8550`

---

## 🎯 Как использовать

### Интерфейс состоит из:
1. **Модель тела (слева)** — интерактивная SVG-схема
   - Наведите курсор на зону → она подсветится
   - Кликните на зону → автоматически выберется упражнение

2. **Панель статистики (справа вверху)** — прогресс по зонам
   - Уровень зоны
   - Прогресс-бар XP
   - Дата последней тренировки

3. **Форма тренировки (справа внизу)**
   - Выберите упражнение (или кликните на тело)
   - Укажите подходы и повторения
   - Нажмите **"⚡ Тренировка"**

### Пример тренировки:
1. Кликните на **Ноги** на модели тела
2. Автоматически выберется "Приседания"
3. Введите: Подходы=3, Повторы=15
4. Нажмите "Тренировка"
5. Получите **+450 XP** (распределённых между Ногами, Прессом и Спиной)

## 🧮 Математика системы

### 1. Прогрессия уровня зоны
```
XP_порог(уровень) = 100 × (уровень ^ 1.8)
```

| Уровень | Нужно XP | Всего XP |
|---------|----------|----------|
| 1→2 | 100 | 0 |
| 2→3 | 348 | 100 |
| 3→4 | 714 | 448 |
| 4→5 | 1,196 | 1,162 |
| 5→6 | 1,812 | 2,358 |
| 10→11 | 6,310 | 19,473 |
| 20→21 | 24,595 | 152,914 |

### 2. Общий уровень (гармоническое среднее)
Штрафует за дисбаланс! Формула:
```
Общий = n / (1/L₁ + 1/L₂ + ... + 1/Lₙ)
```

**Примеры:**
- Все зоны по 50 → **Общий = 50** ✅
- [100, 1, 50, 50, 50, 50, 50, 50] → **Общий = 7** ⚠️
- [10, 10, 10, 10, 10, 10, 10, 10] → **Общий = 10** ✅

### 3. XP за упражнение
```
Базовый XP = Подходы × Повторы × 10

Для каждой зоны:
XP_зоны = Базовый XP × Коэффициент_зоны
```

**Пример (Приседания 3×15):**
- Базовый XP = 3 × 15 × 10 = 450
- Ноги: 450 × 1.0 = **450 XP**
- Пресс: 450 × 0.4 = **180 XP**
- Спина: 450 × 0.2 = **90 XP**

### 4. Декремент (потеря прогресса при простое)
```
Период_полураспада = 7 + Уровень × 0.5 (дней)

Новый_XP = Старый_XP × 0.5^(дни_простоя / период_полураспада)

Минимум = 10% от XP предыдущего уровня (защитный буфер)
```

**Пример:**
- Уровень 10, XP = 10,000
- Пропуск 14 дней
- Период полураспада = 7 + 10×0.5 = 12 дней
- Новый XP = 10,000 × 0.5^(14/12) ≈ **4,719 XP**
- Минимум (10% от ур.9) = ~800 XP → **Останется 4,719 XP**

## ⚙️ Настройка параметров

Все константы находятся в начале файла `gui_app.py`:

### Изменить скорость прокачки:
```python
def get_xp_threshold(level: int) -> int:
    return int(100 * (level ** 1.8))  # Увеличьте 100 для замедления
```

### Изменить чувствительность декремента:
```python
def apply_decrement(...):
    half_life = 7 + level * 0.5  # Уменьшите 0.5 для faster decay
```

### Добавить новую зону:
```python
BODY_ZONES = {
    ...
    "neck": {"name": "Шея", "color": "#FF00FF"},
}
```

### Добавить упражнение:
```python
EXERCISES_DB.append({
    "name": "Новое упражнение",
    "zones": ["legs", "core"],
    "coeffs": [1.0, 0.5]
})
```

## 🔧 Решение проблем

### Ошибка "ModuleNotFoundError: No module named 'flet'"
```bash
pip install flet
```

### Приложение не открывается в браузере
1. Проверьте, что порт 8550 свободен
2. Откройте вручную: `http://localhost:8550`

### Данные не сохраняются
Проверьте права доступа к папке или наличие `user_progress.json`

### Слишком быстрая/медленная прокачка
Измените коэффициент в функции `get_xp_threshold()` (см. выше)

## 📊 Советы по балансу

### Для новичков:
- Начните с 2-3 зон, не пытайтесь качать всё сразу
- Фокусируйтесь на базовых упражнениях (приседания, отжимания, тяги)
- Целевой темп: 2-3 тренировки в неделю

### Для продвинутых:
- Следите за общим уровнем — он упадёт при игноре слабых зон
- Используйте периодизацию: 3 недели работы + 1 неделя разгрузки
- Декремент начнёт влиять после 3+ дней пропуска

### Масштабирование до 200+ уровня:
- Текущая формула поддерживает бесконечный рост
- На уровне 200 нужно ~2.8 миллиона XP в зоне
- Рекомендуется добавить престиж-систему после 100 уровня

## 📞 Поддержка и расширение

### Добавить свою логику:
1. Создайте файл `custom_exercises.py`
2. Импортируйте в `gui_app.py`:
   ```python
   from custom_exercises import MY_EXERCISES
   EXERCISES_DB.extend(MY_EXERCISES)
   ```

### Интеграция с умными часами:
- Экспортируйте данные из `user_progress.json`
- Парсите через Python-скрипт или отправляйте на сервер

### Мультиплеер/соревнования:
- Синхронизируйте JSON-файлы через облако
- Добавьте таблицу лидеров в `gui_app.py`
