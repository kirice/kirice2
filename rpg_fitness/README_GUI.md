# 🎮 RPG Fitness Tracker — GUI Приложение

## 📦 Что это?
Интерактивное приложение для отслеживания тренировок с:
- **Интерактивной моделью тела** (SVG) с подсветкой зон при наведении
- **Плавными анимациями** прогресса и уровней
- **Автоматическим расчетом** XP и уровней
- **Тёмной темой** в стиле киберпанк/RPG
- **Веб-интерфейсом** (открывается в браузере автоматически)

---

## 🚀 Как запустить

### Вариант 1: Через скрипт (рекомендуется)
```bash
cd /workspace/rpg_fitness
./run.sh
```
Приложение откроется в вашем браузере автоматически!

### Вариант 2: Прямой запуск Python
```bash
python gui_app.py
```

### Вариант 3: Создать ярлык на рабочем столе (Linux)
```bash
# Копирование ярлыка в систему
sudo cp rpg_fitness.desktop /usr/share/applications/
# Или в домашнюю папку
cp rpg_fitness.desktop ~/.local/share/applications/
```

После установки ярлыка приложение появится в меню приложений.

---

## 🎯 Как пользоваться

1. **Запустите приложение** — оно откроется в браузере
2. **Наведите курсор** на часть тела (нога, рука, грудь, пресс) — зона подсветится, а остальные затемнятся
3. **Кликните** на зону — автоматически выберется упражнение для неё
4. **Заполните форму**:
   - Выберите упражнение из списка (или оно уже выбрано после клика)
   - Укажите подходы и повторения
   - Нажмите "Записать тренировку"
5. **Наблюдайте** за ростом уровней и плавной анимацией прогресс-баров!
6. При повышении уровня появится **Level Up! уведомление**

## 🎨 Особенности интерфейса

| Элемент | Описание |
|---------|----------|
| **Силуэт человека** | Интерактивная SVG-модель, зоны реагируют на наведение мышью |
| **Прогресс-бары** | Плавная 300ms анимация заполнения XP |
| **Level Up!** | Всплывающее модальное окно при повышении уровня |
| **Общий уровень** | Рассчитывается по гармоническому среднему (штраф за дисбаланс) |
| **Цветовая кодировка** | Каждая зона имеет свой цвет для наглядности |
| **Тёмная тема** | Приятная для глаз, в стиле RPG-игр |

## ⚙️ Настройка

Все параметры можно изменить в начале файла `gui_app.py`:

```python
BODY_ZONES = {
    "legs": {"name": "Ноги", "color": "0xFF5733"},
    # Добавьте свои зоны...
}

EXERCISES_DB = [
    {"name": "Приседания", "zones": ["legs", "core"], "coeffs": [1.0, 0.4]},
    # Добавьте свои упражнения...
]
```

Для расширения базы упражнений используйте полный список из `exercises_db.py`.

```bash
pip install flet flet-web
```

## 💡 Советы

- **Регулярность**: Записывайте тренировки каждый день для максимального прогресса
- **Баланс**: Качайте все зоны равномерно, иначе общий уровень будет низким (гармоническое среднее!)
- **Отдых**: Система учитывает дни отдыха (декремент прогресса при простое от 3 дней)
- **Веб-режим**: Приложение работает в браузере — можно открыть на телефоне или планшете

---

## 🐛 Если что-то не работает

1. Убедитесь, что установлен Python 3.10+: `python --version`
2. Проверьте установку flet: `pip show flet`
3. Переустановите зависимости: `pip install --upgrade flet flet-web`
4. Проверьте, что браузер открывается автоматически (порт может быть занят)
5. Для консольной версии используйте: `python cli.py`
